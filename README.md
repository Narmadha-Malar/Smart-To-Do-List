# Smart To-Do List with Reminders 📝⏰

A task management app with reminder alerts, priority levels and filter functionality built using HTML, CSS and JavaScript.

## About the Project
This app helps users manage their daily tasks with smart reminders. Users can set interval-based or time-based alerts for any task so they never forget important activities.

## Features
- Add, Edit and Delete tasks
- Set High, Medium, Low priority
- Reminder alert - Every X hours
- Reminder alert - At specific time
- Filter tasks - All, Pending, Done
- Task completion tracking
- Stats - Total, Done, Pending count

## Technologies Used
- HTML
- CSS
- JavaScript (setInterval, setTimeout)

## How it Works
1. Add a task with priority level
2. Set reminder - Every X hours or at specific time
3. Alert popup appears when reminder time comes
4. Mark task as done to stop reminder

## Example Use Cases
- Drink Water - Every 1 hour reminder
- Take Medicine - At 2:00 PM reminder
- Exercise - Every 24 hours reminder
